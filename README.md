# 🥗 NutriScan Pro – Smart Diet & Nutrition Analyzer

🔗 **Live Demo:** https://alwaysdaya123.github.io/nutriscan-pro/

---

## 📌 Project Overview

**NutriScan Pro** is an AI-powered web application that helps users analyze food, estimate calories, and track their daily nutrition.

Users can upload food images, view nutritional insights, and monitor their eating habits in a simple, interactive way. The application is designed as a **smart nutrition assistant** to promote healthy lifestyle choices.

---

## 🚀 Features

### 🍽️ Food Analysis
- Upload or capture food images
- Detect food items using AI/ML models
- Display estimated calories and nutrition values

### 📊 Nutrition Insights
- Shows:
  - Calories  
  - Protein  
  - Carbohydrates  
  - Fat  
  - Fiber  
  - Sodium  
- Includes visual charts and progress indicators

### ❤️ Health Score
- Calculates a **health score (0–100)**
- Displays with color indicators and charts
- Helps users evaluate food quality

### 🧾 Food History
- Stores previously analyzed meals
- Displays:
  - Food image  
  - Name  
  - Calories  
  - Date & time  

### 📅 Weekly Dashboard
- Tracks:
  - Total calories consumed  
  - Nutrient distribution  
  - Average health score  
- Includes bar and pie charts

### 🎯 Daily Goal Tracker
- Set daily calorie goals
- Track progress visually

### 🌗 Light & Dark Mode
- Toggle between themes
- Improves accessibility and user experience

### 🔐 Authentication
- Email & password login system
- Secure and simple user access
